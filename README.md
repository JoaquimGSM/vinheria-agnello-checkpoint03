Este projeto foi desenvolvido em JavaScript, HTML e CSS, com o intuito de praticar o uso de arrays, objetos e métodos como: `forEach`, `filter`, `reduce` e `map`.

O sistema permite:

- Criar uma lista de vinhos com as seguintes informações: nome, tipo, safra e estoque.
- Adicionar novos vinhos ao array.
- Listar todos os vinhos cadastrados.
- Verificar vinhos com estoque baixo.
- Calcular o estoque total da vinícola.
- Exibir os nomes dos vinhos em caixa alta.
